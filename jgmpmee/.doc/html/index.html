Java GMP Modular Exponentiation Extension

The Java GMP Modular Exponentiation Extension (JGMPMEE) is a Java wrapper that allows invoking some of the functionality in Gnu Multiprecision Library (GNU MP) and GMPMEE. It was written by Douglas Wikström to speed up some operations in the Vfork library. GMPMEE is licensed under the GNU Lesser General Public License (LGPL).
