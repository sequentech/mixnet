GMP Modular Exponentiation Extension

The GMP Modular Exponentiation Extension (GMPMEE) is a minor extension of the Gnu Multiprecision Library (GNU MP). It adds simultaneous modular exponentiation and fixed base modular exponentiation functionality to the set of integer functions. It was written by Torbjörn Granlund and Douglas Wikström to speed up some operations in the Vfork library. GMPMEE is licensed under the GNU Lesser General Public License (LGPL).
